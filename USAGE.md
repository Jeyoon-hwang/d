# 📖 LOL AI Assistant - 사용 가이드

## 🚀 빠른 시작

### 1. 설치 방법

#### 방법 A: EXE 파일 다운로드 (권장)
1. [Releases](https://github.com/yourusername/lol-ai-assistant/releases) 페이지에서 최신 버전 다운로드
2. `LOL_AI_Assistant_Setup.exe` 실행
3. 설치 마법사 따라하기
4. 바탕화면 아이콘 클릭하여 실행

#### 방법 B: 소스 코드에서 실행
```bash
# 1. 저장소 클론
git clone https://github.com/yourusername/lol-ai-assistant.git
cd lol-ai-assistant

# 2. 의존성 설치
pip install -r requirements.txt

# 3. 실행
python main.py
```

### 2. Riot API 키 설정

1. 애플리케이션 실행
2. **⚙️ 설정** 탭으로 이동
3. [Riot Developer Portal](https://developer.riotgames.com/) 방문
4. 로그인 후 "Development API Key" 발급 (24시간 유효)
5. 발급받은 키를 입력란에 붙여넣기
6. **💾 설정 저장** 클릭

## 📊 주요 기능 사용법

### 🏠 대시보드
- 시스템 개요 및 통계 확인
- 빠른 액션 버튼으로 원하는 기능으로 바로 이동

### 📊 리플레이 분석

**목적**: 소환사의 최근 게임을 분석하여 개선점 찾기

**사용 방법**:
1. **📊 리플레이 분석** 탭 클릭
2. 소환사명 입력 (예: Hide on bush, Faker)
3. **🚀 분석 시작** 클릭
4. 분석 결과 확인:
   - 로밍 패턴
   - 포지셔닝 점수 (초반/중반/후반)
   - CS 효율성 (분당 CS, 효율성 %)
   - 개선 포인트

**분석 지표 해석**:
- **포지셔닝 점수** (1-10)
  - 8-10: 매우 안전 ✓
  - 5-7: 적정
  - 3-4: 위험 ⚠️
  - 1-2: 매우 위험 ✗

- **CS 효율성** (%)
  - 90-100%: 챌린저급 🏆
  - 80-89%: 다이아급
  - 70-79%: 플레티넘급
  - 60-69%: 골드급
  - 60% 미만: 개선 필요 📈

### ⚔️ 챔피언 정보

**목적**: 각 챔피언의 스킬, 상성, 파워스파이크 학습

**사용 방법**:
1. **⚔️ 챔피언 정보** 탭 클릭
2. 드롭다운에서 챔피언 선택
3. 정보 확인:
   - **스킬 정보**: Q, W, E, R 스킬 설명
   - **능력치**: 로밍/웨이브클리어/폭딜/기동성/CC
   - **파워스파이크**: 강력해지는 레벨
   - **코어 아이템**: 필수 아이템 빌드
   - **상성**: 유리/불리한 챔피언

**로밍 TOP 5**: 로밍에 가장 강한 챔피언 확인

### 🎯 실시간 조언

**목적**: 현재 게임 상황에 맞는 최적의 행동 추천

**사용 방법**:
1. **🎯 실시간 조언** 탭 클릭
2. 게임 정보 입력:
   - **내 챔피언**: 플레이 중인 챔피언 선택
   - **게임 시간**: 현재 게임 경과 시간 (분)
   - **내 레벨**: 현재 레벨
3. **🎯 AI 조언 받기** 클릭
4. AI 추천 확인:
   - **현재 상태 분석**
     - 웨이브 상태 (푸시/프리즈)
     - 맵 포지션 (안전/위험)
     - 파워 레벨 (유리/불리)
   - **추천 행동** (우선순위순)
   - **로밍 분석**
   - **오브젝트 우선순위**

**실시간 조언 활용 팁**:
- 게임 시작 전/후에 조언 받기
- 중요한 타이밍(6레벨, 아이템 완성)에 확인
- 오브젝트 스폰 전에 미리 확인
- 딜교환 전에 파워 레벨 체크

## 🎯 AI 추천 시스템 이해하기

### 추천 행동 우선순위

**우선순위 10 (긴급)**:
- 위험 상황 → 즉시 후퇴
- 바론 타이밍

**우선순위 8-9 (매우 중요)**:
- 드래곤 싸움
- 올인 가능 상황
- CS 매우 부족

**우선순위 6-7 (중요)**:
- 로밍 추천
- 딜교환 유리
- 웨이브 관리

**우선순위 5 이하 (일반)**:
- 파밍
- 시야 확보
- 리콜

### 게임 페이즈별 전략

**초반 (0-15분)**:
- CS 확보 최우선
- 안전한 포지셔닝
- 파워 스파이크 도달 집중
- 와드 설치

**중반 (15-30분)**:
- 로밍으로 영향력 확대
- 오브젝트 싸움 참여
- 시야 싸움
- 팀플레이 중시

**후반 (30분+)**:
- 한타 승리 최우선
- 바론/엘더 드래곤 확보
- 포지션 실수 금지
- 팀원과 함께 움직이기

## 🔧 문제 해결

### API 키 오류
**증상**: "API 키가 유효하지 않습니다"
**해결**:
1. API 키가 올바르게 입력되었는지 확인
2. 24시간이 지나지 않았는지 확인
3. 새 API 키 발급 후 재입력

### 소환사를 찾을 수 없음
**증상**: "소환사를 찾을 수 없습니다"
**해결**:
1. 소환사명 정확히 입력 (띄어쓰기 주의)
2. 서버 지역 확인 (기본: 한국)
3. 최근 게임이 있는지 확인

### 프로그램이 느림
**해결**:
1. API 요청 수 제한 (분당 20회)
2. 동시에 여러 분석 실행하지 않기
3. 인터넷 연결 확인

### 프로그램이 실행되지 않음
**해결**:
1. Windows Defender 예외 처리
2. 관리자 권한으로 실행
3. Visual C++ Redistributable 설치
